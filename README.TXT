This project is a simple http server implemented with Netty framework. 
Created as @HamsterCoders assignment.
Author: @Fleischers.

#Basic features:
	- Asynchronous I/O
	- About 10000? queries per second

#How to build:
	- Maven? 

#How to run:
	##Server side use
		Run in command line "java -jar output\HCServer.jar" and server will be launched at http://localhost:8080/ 
		or simply use "launchHCServer.bat" if using OS Windows.
		For normal server work port 8080 is needed to be free.
		*if you want to specify another port you can add it as parameter e.g. "java -jar output\HCServer.jar 8081"
	##Client side use
		Go to browser at http://localhost:8080/ and follow the instructions
